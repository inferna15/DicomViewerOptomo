# DicomViewerOptomo
ActiViz .NET kullanarak VTK'yı C# dilinde kullanarak yaptığım bir dicom görüntüleme arayüzü. Genel mantığı, dicom görüntüleri bir bütün haline getirip X, Y ve Z eksenlerinde görüntülememizi sağlıyor. Bu uygulamada en çok uğraştığım şey eksen çizgilerinin doğru şekilde çalışmasıydı.
